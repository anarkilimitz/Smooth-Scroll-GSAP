# Smooth Scroll как в GSAP (vanilla JS)

Лёгкая имитация очень плавного скролла в стиле GSAP ScrollSmoother / Lenis / Locomotive Scroll — **без каких-либо библиотек**, только чистый JavaScript + CSS.

## Демо

[Открыть демо →](https://anarkilimitz.github.io/Smooth-Scroll-GSAP/)

## Особенности

- Очень мягкий и приятный скролл  
- Работает через `position: fixed` + `transform: translateY`  
- Поддерживает изменение размера окна  
- Минимальный код (~50 строк логики)  
- Использует `requestAnimationFrame` и `passive: true`  
- Не требует сторонних зависимостей
